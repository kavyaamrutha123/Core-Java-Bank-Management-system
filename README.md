Bank Management System

A console-based banking application developed using Core Java. This project allows users to perform basic banking operations such as account management, deposit, withdrawal, and balance inquiry.

Features

View Account Details
Deposit Money
Withdraw Money
Check Account Balance
Input Validation
Menu-Driven Interface

Technologies Used

Core Java
OOP Concepts
Exception Handling
Scanner Class

OOP Concepts Implemented

 Encapsulation
 Classes and Objects
 Constructors
 Methods

Project Outcome

This project demonstrates the implementation of fundamental banking operations while applying Core Java and Object-Oriented Programming principles.
